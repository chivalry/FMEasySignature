Using the Demo Database

FMEasySignature can be used as a local database, shared using peer-to-peer networking, or hosted on a server running FileMaker Server 13 or later.

FMEasySignature is compatible with FileMaker Pro, FileMaker Pro Advanced, and FileMaker Go.
